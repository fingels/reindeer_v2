# Dependencies

For Cmake:
```
sudo apt-get -y install cmake
```

# Usage

## Build

If necessary, `mdir build` and then:

```
cd build
cmake ..
cmake --build .
```

## Usage

The executable is a dummy code while the CMake is configured properly. Execute it while in `build` folder:

```
./app/reindeer2
```
